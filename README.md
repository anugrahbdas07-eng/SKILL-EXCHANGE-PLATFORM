**Skill Exchange** is a high-performance, single-page Learning Management System (LMS) designed to deliver interactive technical education without the need for a complex backend. 

🔗https://skill-exchange-platform-rouge.vercel.app/

## ✨ Core Features

* **Interactive Split-Screen Workspaces:** Integrated live compilers (via OneCompiler) and professional design tools (Photopea) directly into the browser.
* **Rigorous Assessment Engine:** Built-in 5x5 module quizzes with strict grading logic (requires 60% to pass) to ensure concept mastery.
* **Dynamic Certificate Generation:** Automatically generates a professional, rigorously styled certificate with a randomized verification ID and one-click PDF download using the browser's native print engine.
* **Live Search & Filtering:** A Google-style live dropdown search bar to instantly navigate through 10+ professional tracks.
* **Interactive AI Assistant UI:** A sleek, logic-based chatbot interface built seamlessly into the platform to guide users.
* **Glassmorphism UI:** Modern, high-contrast design featuring dark-tinted glass panels over a CSS-animated live background.

## 🛠️ Technical Implementation
This project showcases advanced DOM manipulation and Single Page Application (SPA) architecture using purely vanilla HTML, CSS, and JavaScript. 
* All state management (course progression, grading, UI toggling) is handled entirely via client-side JavaScript.
* Designed for instantaneous load times and deployed continuously via **Vercel**.

## 🚀 How to Run Locally
1. Clone this repository: `git clone https://github.com/anugrahbdas07-eng/SKILL-EXHCNAGE-PLATFORM.git`
2. Open the folder in VS Code.
3. Open `index.html` in your web browser (or use the VS Code Live Server extension). No server installation required!
